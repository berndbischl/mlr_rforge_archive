Training

"Training" a learner just means fitting a model to a given data set. We are not concerned with the specifics of the fitting process as such here - this will be taken care of by the underlying R methods that this package integrates. Rather more important is that training and all subsequent operations can be performed by using a unified interface.

This is in this case achieved by calling the method train on the classification task. The classification or regression method can be specified either by passing its name as a string or by passing a Learner object. While the former is maybe more covenient, the latter allows for more flexibility, for example it permits setting the hyperparameters of the learning methods. Optionally, the indices of the subset which are used for training can be passed via the subset argument.

The return value is always an object of class Model which wraps the concrete model of the used R classification or regression method. It can subsequently be used to perform a prediction for new observations.

Classification example

Let's have a look at the iris data set. 

	# Create the classification task. 
	ct <- makeClassifTask(data = iris, target = "Species")
	
	# Let's train some Decision Trees.
	#   On the whole data set.
	m <- train("classif.rpart", ct)
 
	#   On a subset (every second observation).
	m <- train("classif.rpart", ct, subset = seq(from = 1, to = 150, by = 2))

	#   In order to use non-default values of hyperparameters a Learner object has to be created before training.
	l <- makeLearner("classif.rpart", minsplit = 7, cp = 0.03)
	m <- train(l, ct, subset = seq(from = 1, to = 150, by = 2))
	
	# You can print some basic information of the model to the console.
	m

Regression example

As regression example we use the BostonHousing data set. 

	# Create the regression task. 
	library(mlbench); data(BostonHousing)
	rt <- makeRegrTask(data = BostonHousing, target = "medv")
	
	# Let's train some Gradient Boosting Machine.
	#   On the whole data set.
	m <- train("regr.gbm", rt) 
	
	#   On a subset (every second observation).
	m <- train("regr.gbm", rt, subset = seq(1, 506, 2))
	
	#   In order to use non-default values of hyperparameters a Learner object has to be created before training.
	l <- makeLearner("regr.gbm", n.trees = 500, distribution = "laplace", interaction.depth = 3)
	m <- train(l, rt, subset = seq(1, 506, 2))	
	
	# You can print some basic information of the model to the console. 
	m
	
	Learner model for id=regr.gbm class=regr.gbm
	Trained on obs: 253
	Used features: 13
	Hyperparameters: distribution=laplace,n.trees=500,interaction.depth=3
