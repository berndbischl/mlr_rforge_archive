Quickstart

This is quickstart gives you a first glance at the features ans usage of the mlr package. All shown functions are explained in detail at the following chapters of this tutorial.

We will start with a simple training example on the iris data set followed by a validation of the classification with a cross-validation.

	# First we create a classification task for the iris data set with the species as target.
	ct <- makeClassifTask(data = iris, target = "Species")
	 # Now we train a random forest on the whole data set
	m <- train(learner="classif.randomForest",task=ct)
	# We can now for example access the model.
	m@learner.model
	# Now we will validate our model with a 10-fold cross-validation
	# First a new resample description, which can be used with an given task.
	rdesc <- makeResampleDesc("CV", iters = 10)
	# And finaly we start the resampling with a specific learner.
	res <- resample(learner="classif.randomForest", task="ct", resampling="rdesc")
