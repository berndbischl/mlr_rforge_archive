learn.task-class {mlr}R Documentation

Base class for learning tasks.

Description

A learning task is a general description object for a machine learning experiment. It wraps the data source and specifies - through its subclasses - the type of the task (e.g. classification or regression), the target variable and other details of the problem. As this is just an abstract base class, you should not instantiate it directly but use the inheriting classes and their factory methods.
