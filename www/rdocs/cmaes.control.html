cmaes.control {mlr}R Documentation

Control for CMA-ES tuning.

Description

Control structure for CMA-ES tuning.

Usage

cmaes.control(start, minimize=T, scale=identity, ...)

Arguments

start [numeric]
Named vector of initial values.
minimize [logical]
Minimize performance measure? Default is TRUE.
scale [function]
A function to scale the hyperparamters. E.g. maybe you want to optimize in some log-space. Has to take a single, numerical vector and return a scaled one. Default is identity function.
... Passed down to cmaes.

Details

See cma_es for details of CMA-ES settings.

Value

Control structure for tuning.
