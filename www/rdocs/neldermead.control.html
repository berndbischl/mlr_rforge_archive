neldermead.control {mlr}R Documentation

Control for Nelder-Mead tuning.

Description

Control structure for Nelder-Mead tuning.

Usage

neldermead.control(start, minimize=T, scale=identity, ...)

Arguments

start [numeric]
Named vector of initial values.
minimize [logical]
Minimize performance measure? Default is TRUE.
scale [function]
A function to scale the hyperparameters. E.g. maybe you want to optimize in some log-space. Has to take a single, numerical vector and return a scaled one. Default is identity function.
... Passed down to optim.

Details

See optim for details of Nelder-Mead settings.

Value

Control structure for tuning.
