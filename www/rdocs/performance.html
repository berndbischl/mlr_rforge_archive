performance {mlr}R Documentation

Measure performance and losses of prediction.

Description

Measures the quality of predictions w.r.t. some performance measures or loss functions.

Usage

performance(pred, measures, aggr, losses, task)

Arguments

pred [prediction]
Prediction object to evaluate.
measures [see measures] Performance measures.
aggr [see aggregations] Aggregation functions. Ignored if not a resample.prediction
losses [see losses] Loss functions.
task [learn.task]
Optionally specifies learning task, very rarely needed.

Value

performance-methods: A list with with possibly three named components: "measures" is a data.frame of performance values, "aggr" a data.frame of aggregated values, "losses" a data.frame of losses.
