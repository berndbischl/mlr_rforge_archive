conf.matrix {mlr}R Documentation

Confusion matrix.

Description

Calculates confusion matrix for (possibly resampled) prediction.

Usage

conf.matrix(result, relative)

Arguments

result [prediction]
Result of a prediction.
relative [logical]
If TRUE rows are normalized to show relative frequencies.

Value

A confusion matrix.
