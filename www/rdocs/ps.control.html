ps.control {mlr}R Documentation

Control for pattern search tuning.

Description

Control structure for pattern search tuning.

Usage

ps.control(start, lower=rep(-Inf, length(start)), upper=rep(Inf, length(start)),
    delta=1, delta.min=10^(-3), maxit=100, minimize=TRUE,
    scale=identity)

Arguments

start [numeric]
Named vector of initial values.
lower [numeric]
Named vector of lower boundary constraints. Default is -Inf.
upper [numeric]
Named vector of upper boundary constraints. Default is Inf.
delta [numeric]
Initial step size.
delta.min [numeric]
Minimal step size. Default is 10^-3.
maxit [integer]
Maximal number of iterations. Default is 100
minimize [logical]
Minimize performance measure? Default is TRUE.
scale [function]
A function to scale the hyperparameters. E.g. maybe you want to optimize in some log-space. Has to take a single, numerical vector and return a scaled one. Default is identity function.

Value

Control structure for tuning.
