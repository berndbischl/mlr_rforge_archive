ROCR.plot.taskR Documentation

Compare ROC curves of learners in a benchmark experiment.

Description

Compare ROC curves of learners in a benchmark experiment.

Usage

ROCR.plot.task(x, task.id, learner.ids=x["learners"], perf1="tpr", perf2="fpr",
    legend.x="bottomright", legend.y, col, ...)

Arguments

x

[bench.result]
Benchmark result. Must contain predictions.

task.id

[string]
Id of a task from x.

learner.ids

[character]
Ids of learners to plot. Default are all learners in x for selected task.

perf1

[string]
First ROC measure for y-axis. Note that this must be a ROCR measure, see ROCR.performance. Default is "tpr".

perf2

[string]
Second ROC measure for x-axis. Note that this must be a ROCR measure, see ROCR.performance. Default is "fpr".

legend.x

[any]
Where should legend be placed. Any placement accepted by legend is ok. "none" does not plot a legend.

legend.y

[any]
Where should legend be placed. Any placement accepted by legend is ok.

col

[character]
Colors for ROC curves. Default are rainbow colors.

...

[any]
Further arguments that are passed to ROCR.plot.performance.
