makeResampleInstanceR Documentation

Construct resampling instance

Description

Generates an instance object for a resampling strategy.

Arguments

desc

[ResampleDesc]
Resampling description.

task

[LearnTask]
Data of task to resample from. Prefer to pass this instead of size.

size

[integer]
Size of the data set to resample. Can be used instead of task.

Value

makeResampleInstance-methods: A ResampleInstance object.

See Also

makeResampleDesc, resample
