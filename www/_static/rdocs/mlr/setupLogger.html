setupLoggerR Documentation

Logger setup.

Description

Sets up the logging system of mlr.

Usage

setupLogger(console=TRUE, file=NA, level, sublevel=NA)

Arguments

console

[logical(1)]
Should output be printed to R console?

file

[character(1)]
Path to file to redirect output into.

level

[character]
Which logging levels should be printed: 'error', 'warn', 'info', 'debug'. Default is 'info'.

sublevel

[character]
Which logging sublevels should be printed. Default is NA which means all logging of the selected main level is printed. Currently for 'debug' are available: 'train', 'predict' and 'parallel'. Currently for 'info' are available: 'tune'.

Value

NULL.
