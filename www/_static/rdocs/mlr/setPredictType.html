setPredictTypeR Documentation

Set predict type of learner object.

Description

Set the type of prediction the learner should return.

Arguments

learner

[Learner]
Learner object.

predict.type

[character(1)]
Classification: “response” or “prob”, specifying the type to predict. Default is “response”. Ignored for regression.

Details

setPredictType-methods: Possible prediction types are labels / numeric response or class probabilities.

Value

setPredictType-methods: Learner with changed Prediction behaviour.

See Also

setThreshold to alter the threshold used for prediction.
