TaskDesc-classR Documentation

Description object for task.

Description

Description object for task, encapsulates basic statistics without having to store the complete data set.

Slots

type:

(character) Type of task, either “classif” for classification or “regr” for regression.

id:

(character) Id string of task.

target:

(character) Name of target variable.

size:

(integer) Number of cases.

n.feat:

(integer) Number of covariates, named vector with entries: “numerics”, “integers”, “factors”, “characters”, “logicals”.

class.levels:

(character) All possible classes. Character vector. NA if not classification.

has.missing:

(logical) Are missing values present?

has.inf:

(logical) Are infinite numerical values present?

has.weights:

(logical) Are weights available in task for covariates?

has.blocking:

(logical) Is blocking available in task for observations?

positive:

(character) Positive class label for binary classification, NA else.

negative:

(character) Negative class label for binary classification, NA else.
