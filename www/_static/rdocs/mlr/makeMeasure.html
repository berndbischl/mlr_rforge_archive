makeMeasureR Documentation

Construct your own performance measure.

Description

Construct your own performance measure.

Arguments

id

[character(1)]
Name of measure.

minimize

[logical(1)]
Should the measure be minimized? Default is TRUE.

classif

[logical(1)]
Is the measure applicable for classification? Default is FALSE.

regr

[logical(1)]
Is the measure applicable for regression? Default is FALSE.

only.binary

[logical(1)]
Is the measure only applicable to binary classification? Only reasonable if classif is TRUE. Default is FALSE.

allowed.pred.types

[character] Which prediction types are allowed for this measure? Subset of “response”,“prob”. Default is character(0).

fun

[function(task, model, pred, extra.args)]
Calculates performance value.

extra.args

[list]
List of extra arguments which will always be passed to fun.

Value

makeMeasure-methods: Measure

See Also

measures

Examples

f <- function(task, model, pred, extra.args) sum((pred@df$response - pred@df$truth)^2) 
makeMeasure(id="my.sse", minimize=TRUE, regr=TRUE, allowed.pred.types="response", fun=f)
