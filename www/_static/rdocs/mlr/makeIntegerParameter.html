makeIntegerParameterR Documentation

Integer variable for optimization.

Description

Integer variable for optimization.

Usage

makeIntegerParameter(id, lower=-.Machine$integer.max, upper=.Machine$integer.max,
    trafo=identity)

Arguments

id

[character(1)] Name of parameter.

lower

[numeric(1)]
Lower bound. Default is -Inf.

upper

[numeric(1)]
Upper bound. Default is Inf.

trafo

[function(x)]
Function to transform parameter. Is applied to the parameter before it is passed to its corresponding fitness function. Default is identity.

Value

Parameter
