make.res.instanceR Documentation

Construct resampling instance

Description

Generates an instance object for a resampling strategy.

Arguments

x

[string or resample.desc]
"cv" for cross-validation, "stratcv" for stratified cross-validation, "repcv" for repeated cross-validation,
"bs" for out-of-bag bootstrap, "bs632" for B632 bootstrap, "bs632plus" for B632+ bootstrap,
"subsample" for subsampling, "holdout" for holdout.

task

[integer]
Data of task to resample from. Prefer to pass this instead of size.

size

[integer]
Size of the data set to resample.

iters

[integer]
Number of resampling iterations. Not needed for "holdout".

...

[any]
Further parameters for strategies.
split: Percentage of training cases for "holdout", "subsample".
reps: Repeats for "repcv"

Value

make.res.instance-methods: A resample.instance object.
