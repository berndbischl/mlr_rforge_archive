makeIntegerVectorParameterR Documentation

Numeric vector variable for optimization.

Description

Numeric vector variable for optimization.

Usage

makeIntegerVectorParameter(id, dim, lower=-.Machine$integer.max, upper=.Machine$integer.max,
    trafo=identity)

Arguments

id

[character(1)] Name of parameter.

dim

[integer(1)]
Length of vector.

lower

[numeric(dim)]
Lower bound. Default is -Inf.

upper

[numeric(dim)]
Upper bound. Default is Inf.

trafo

[function(x)]
Function to transform parameter. Is applied to the parameter before it is passed to its corresponding fitness function. Default is identity.

Value

Parameter
