getPropertyR Documentation

Get property of learner object.

Description

Get property of learner object. Currently available are:

type [character(1)]

Can be “classif” or “regr”.

numerics [logical(1)]

Can numeric features be handled?

factors [logical(1)]

Can factor features values be handled?

missings [logical(1)]

Can missing values be handled?

weights [logical(1)]

Can case weights be handled?

oneclass [logical(1)]

Can one-class problems be handled?

twoclass [logical(1)]

Can two-class problems be handled?

multiclass [logical(1)]

Can multi-class problems be handled?

prob [logical(1)]

Can probabilities be predicted?

Arguments

learner

[Learner]
Learner object.

prop

[character(1)]
Name of property.

Value

getProperty-methods: Property values.

See Also

setProperty, setProperties
