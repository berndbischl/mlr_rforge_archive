Learning tasks

Learning tasks are the basic elements of the package to encapsulate the data set and off informtaion regarding the target of the modelling process.

Classification example

	ct <- make.classif.task(data=iris, target="Species")

This single line already performs quite a few task:

  • Tries to load required package for LDA (MASS)
  • Gathers information about features of data set and method, and compares whether they are compatible (e.g. some methods might not handle NAs or facors)
  • Converts integers to numerics and warns about this.

The template also gives you some convenience methods to access properties of the data set and the machine learning method:
