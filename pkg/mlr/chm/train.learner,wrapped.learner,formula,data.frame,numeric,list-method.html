train.learner

Description

Mainly for internal use. Trains a wrapped learner on a giving training set, possibly w.r.t. some hyperparamters and case weights.

Arguments

wrapped.learner [wrapped.learner]
Wrapped learner from this package.
formula [formula]
Specifies inputs and output.
data [data.frame]
Training set
weights [numeric]
Optional case weights, default is 1.
parset [numeric]
Named list of hyperparameters. Default is empty list.

Value

Model of the underlying learner.
