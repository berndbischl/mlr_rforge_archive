make.subsample.desc

Description

Generates a description object for subsampling. Usually only needed in benchmark to describe the inner resampling.

Usage

make.subsample.desc(size, split=2/3, iters)

Arguments

size [integer]
Size of the data set to resample.
split [numeric]
Proportion of data used for training set. Default is 2/3.
iters [integer]
Number of generated subsets / resampling iterations.

Value

A subsample.desc object.

See Also

subsample.desc, benchmark
